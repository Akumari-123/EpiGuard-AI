# EpiGuard AI 🧠

## 📌 Project Overview

EpiGuard AI is an AI-powered health-tech solution designed to predict and manage the spread of epidemics. It helps users understand diseases, symptoms, precautions, and consequences through an intelligent chatbot and data-driven insights.

---

## 🚀 Features

* 🦠 Disease Prediction System
* 🤖 AI Chatbot for health guidance
* 📊 Firebase-based real-time data handling
* 📢 Awareness about symptoms, prevention & consequences
* 🌐 User-friendly interface for quick access

---

## 🛠️ Tech Stack

* Frontend: Next.js (React)
* Backend: Firebase
* AI Integration: OpenAI / AI APIs
* Styling: Tailwind CSS

---

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/your-username/EpiGuard-AI.git

2. Install dependencies:
   npm install

3. Run the project:
   npm run dev

4. Open in browser:
   http://localhost:3000

---

## 🔄 Technical Workflow

1. User enters disease-related query
2. AI chatbot processes the query
3. Firebase handles data storage and retrieval
4. System provides prediction, precautions, and guidance
