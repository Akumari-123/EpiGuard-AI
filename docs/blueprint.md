# **App Name**: EpiGuard AI

## Core Features:

- Interactive Dashboard: Visualize key epidemic statistics (cases, deaths, recovery), time-series graphs of trends, and comparison charts for vaccination/mobility impact in an easy-to-digest format.
- Epidemic Prediction Module: Allow users to select specific regions and future dates to generate time-series forecasts of case numbers and derive predictive risk levels (Low, Medium, High).
- Dynamic Risk Map: Display an interactive world map highlighting regions based on their predicted outbreak risk using a clear color-coded system (Red for High, Yellow for Medium, Green for Low).
- AI-powered Insights Panel: Generate AI-driven insights and anomaly detection tool for users, revealing the impact of factors like mobility and vaccination on disease spread trends.
- Centralized Data Integration: Integrate and process historical epidemiological data from sources like Johns Hopkins University, Our World in Data, and Google Mobility for analysis and prediction.
- Responsive User Interface: Provide a modern, clean, and responsive design across desktop and mobile devices, utilizing charts, cards, and interactive maps for optimal data visualization.
- Firebase Backend & APIs: Implement a robust Firebase backend with Firestore for data persistence and authentication, managing data updates and serving predictions to the frontend.

## Style Guidelines:

- The chosen colors reflect health, data clarity, and trustworthiness. The scheme is light for readability of dense data, balanced with a calm yet professional primary and an impactful accent.
- Primary color: A deep, reassuring cyan-blue (#338099), evoking professionalism and clarity without being overtly clinical.
- Background color: A very light, almost white, blue-gray (#EBF1F2), providing a clean canvas that visually extends the primary hue's calming effect.
- Accent color: A vibrant violet-blue (#7578E5), analogous to the primary color, used to highlight critical data points, alerts, and calls-to-action, offering a distinct visual contrast.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly readable characteristics, suitable for conveying complex data clearly across all screen sizes.
- Utilize clean, vector-based icons with a slightly rounded aesthetic. Icons should clearly represent epidemiological concepts, data points, and navigation, such as trend arrows, map markers, and dashboard elements.
- Employ a responsive grid-based dashboard layout with clearly defined card components for statistics and modular sections for charts and interactive maps, ensuring adaptability for various devices and data presentation needs.
- Incorporate subtle, smooth transition animations for loading states, data updates within charts, map interactions (e.g., zooming, region highlighting), and navigation between dashboard modules to enhance user experience without distraction.
